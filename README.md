## 🚀 Getting Started  
### Installation  
1. Download Python from [python.org](https://www.python.org/downloads/).  
2. Verify installation:  
   ```bash  
   python --version  
   # Output: Python 3.x.x  
   ```  

### Your First Python Script  
Create `hello.py`:  
```python  
print("Hello, Python! 🎉")  
```  
Run it:  
```bash  
python hello.py  
```
